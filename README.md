# minecraft_webgpu
Minecraft clone in C++ and WebGPU just for fun!  

2023/10/10:  

Features -  
block placing/destroying, SSAO, simple terrain gen, global shadows, translucent water, imgui-integration  
Tested on macOS (m1).
